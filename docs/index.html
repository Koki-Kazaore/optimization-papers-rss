Subscribe in Slack

  1. Open any Slack channel or DM where you want to receive updates.
  2. Type this command and press Enter:
    /feed https://koki-kazaore.github.io/mathematical-optimization-papers-rss/feed.xml

  3. New papers will appear as Slack notifications when the feed is updated.

Alternatively, use the RSS app in Slack (Apps → RSS) and add the feed URL there.
